# Copy Iteration Workflow

How to refine copy output (titles, descriptions, hashtags) based on user feedback. Since copy output is text (not files), iteration means modifying the selected variation inline rather than re-rendering.

## Core Principle

**Refine the selected variation surgically** rather than regenerating all 4 from scratch. The user picked a direction — honor that choice and make targeted improvements.

## Iteration Workflow

1. **User picks a variation** (e.g., "I like V2")
2. **User requests a change** (e.g., "make it shorter", "more professional", "different angle")
3. **Map the request** to a specific modification (see mapping tables in skill-specific iteration refs)
4. **Apply the change** to the selected variation only
5. **Present the revised version** as VX-r2 (variation X, revision 2)
6. **Show the diff** — what changed and why

## Preserve vs Regenerate Rules

### Always Preserve
- The user's chosen variation number (V1, V2, V3, V4)
- The core topic and platform target
- Research insights already gathered
- The structural framework (e.g., if V2 used PAS, keep PAS unless asked to switch)

### Regenerate When
- User says "start over" or "try something completely different"
- User wants a different platform (YouTube → Instagram requires structural change)
- User provides new/different topic information
- The entire approach is wrong (e.g., "this isn't what my video is about")

## Common Refinement Mappings (All Skills)

| User Says | Action |
|-----------|--------|
| "shorter" / "more concise" | Compress to fewer words, tighter phrasing |
| "longer" / "more detail" | Expand with additional context, keywords, or sections |
| "more professional" | Remove urgency words, use authority framing, formal tone |
| "more casual" / "more fun" | Add conversational tone, contractions, colloquialisms |
| "more clickbait" | Add power words, curiosity gap, brackets, urgency |
| "less clickbait" / "more authentic" | Remove hype words, use straightforward language |
| "different angle" | Reframe the hook — same topic, different emotional trigger |
| "different formula" | Swap to a different copywriting framework entirely |
| "add keywords" / "more SEO" | Integrate target keywords naturally into the copy |
| "make it urgent" | Add time pressure, scarcity, FOMO elements |
| "too salesy" | Soften CTAs, lead with value instead of ask |

## Version Naming Convention

- **Original output:** V1, V2, V3, V4
- **Revisions:** V2-r2, V2-r3 (variation 2, revision 2, revision 3)
- **Combined elements:** V5 (new variation combining elements from multiple originals)
- **Platform adaptation:** V2-ig, V2-tt (variation 2 adapted for Instagram, TikTok)

## Presenting Revisions

Always show:
1. **What changed** — brief summary of the modification
2. **The revised text** — full updated copy
3. **Character count** — updated count and platform compliance
4. **Comparison note** — how the revision differs from the original

```
V2-r2 — [Revision Name]
  Change: [what was modified and why]
  "[Updated copy text here]"
  Characters: XX/YY | Above fold: [preview]
```
