# Copy Quality Validation

Validation checklist for all copy output (titles, descriptions, hashtags). Run these checks before presenting variations to the user.

## Universal Checks (All Skills)

### 1. Character Count Compliance
Verify the output fits within platform limits from [platform-specs.md](platform-specs.md):

| Platform | Title Max | Description Max | Hashtag Max Count |
|----------|-----------|----------------|-------------------|
| YouTube | 100 chars | 5,000 chars | 15 (recommend 3-5) |
| Instagram | N/A | 2,200 chars | 30 (recommend 3-5) |
| TikTok | N/A | 4,000 chars | No hard limit (recommend 3-5) |
| Twitter/X | 280 chars | N/A | No limit (recommend 1-2) |
| LinkedIn | N/A | 3,000 chars | No limit (recommend 3-5) |

### 2. Above-the-Fold Hook
- The most compelling element must appear within the platform's above-fold threshold
- YouTube titles: hook in first 50 characters
- YouTube descriptions: value proposition in first 150 characters
- Instagram/LinkedIn: scroll-stopping hook before truncation point

### 3. Platform Formatting
- Hashtags use PascalCase (e.g., `#SocialMediaTips` not `#socialmediatips`)
- YouTube timestamps start at `00:00`, minimum 3, ascending order
- Twitter/X: no hashtag at start of post, max 2 hashtags
- LinkedIn: hashtags at bottom of post
- No banned/inappropriate hashtags

### 4. Readability
- Clear, natural language (no keyword stuffing)
- Grammar and spelling correct
- Tone consistent throughout
- No orphaned sentences or incomplete thoughts

### 5. CTA Presence
- At least one clear call-to-action where appropriate
- CTA matches the platform convention (subscribe, follow, save, comment)
- CTA is natural, not aggressive

## Title-Specific Checks

- [ ] Hook front-loaded in first 50 characters
- [ ] At least one power word or emotional trigger
- [ ] No misleading claims the content can't support
- [ ] Bracket modifier included in at least one variation
- [ ] Listicle titles use odd numbers when possible
- [ ] Not generic — specific enough to stand out
- [ ] Active voice with verbs (not label headlines)

## Description-Specific Checks

- [ ] First 2 lines work as standalone hook (above-fold test)
- [ ] Primary keyword in first sentence (YouTube SEO)
- [ ] Timestamps properly formatted (YouTube only): `MM:SS Chapter Title`
- [ ] CTA present (subscribe, comment, link click, etc.)
- [ ] No wall of text — formatted with line breaks, structure
- [ ] Links/placeholders clearly marked for user to fill in
- [ ] Complements the title (doesn't repeat it verbatim)

## Hashtag-Specific Checks

- [ ] Count within platform limits
- [ ] All PascalCase (accessibility requirement)
- [ ] Mix of broad, medium, and niche (pyramid strategy)
- [ ] No banned or shadowbanned hashtags (flag any suspected ones)
- [ ] Relevant to the content (no off-topic tags for reach)
- [ ] No duplicate hashtags within a set
- [ ] No special characters that break hashtags (spaces, hyphens, periods)
- [ ] YouTube: first 3 hashtags are the most strategic (shown above title)

## Variation Differentiation Check

All 4 variations must differ **meaningfully** — not just word swaps:

- [ ] Each variation uses a genuinely different approach (formula, angle, structure, or emphasis)
- [ ] At least 2 distinct copywriting frameworks represented across the 4 variations
- [ ] Variations serve different strategic purposes (safe bet, risky/high-reward, SEO, mobile)
- [ ] No two variations could be mistaken for minor edits of each other

## Presenting Validation Results

Include these metrics with each variation:

```
V1 — [Name]
  "[Copy text]"
  Characters: XX/YY ✓
  Above fold: "[first N chars preview]" ✓
  Hook: [power word / curiosity gap / specificity] ✓
  Framework: [AIDA / PAS / Hook-Value-CTA]
```

Flag any validation failures with ✗ and explain the issue.
